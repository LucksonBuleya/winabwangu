# README FILE

## HOW TO RUN ON PC (FOR GROUP MEMBERS)

1. After extracting the project, place the entire wina_bwangu folder in:

   C:\xampp\htdocs\ or  (or whichever folder you used for your previous XAMPP projects).

2. Open XAMPP Control Panel and start both Apache and MySQL.

3. Import the database file:

   * Open phpMyAdmin in your browser (`http://localhost/phpmyadmin`).
   * Create a new database named wina_wangu_db.
   * Import the provided SQL file named wina_wangu_db.sql from the project folder.


### Folder Information

* The "Static" subfolder contains the images used in the project.
* The database contains 5 tables, including the "users" table.

### Additional Notes

* The Admin Dashboard has a "CSV button"  (top right) that exports data into a "CSV file".
  A CSV file (“Comma-Separated Values”) is a plain text file that stores tabular data like a spreadsheet (rows and columns).
